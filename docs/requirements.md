# WheelSentinel AI

## Project Vision

WheelSentinel AI is an Industry 4.0 platform inspired by workflows observed during a web development internship at Rail Wheel Factory (RWF), Yelahanka.

The platform demonstrates how AI, predictive analytics, and modern web technologies can support condition-based maintenance for manufacturing equipment through simulated industrial telemetry.

---

# Objectives

- Monitor machine health in real time.
- Simulate industrial sensor telemetry.
- Predict machine failures before breakdown.
- Estimate Remaining Useful Life (RUL).
- Assist maintenance engineers using AI.
- Track maintenance activities.
- Visualize plant health through dashboards.
- Generate actionable analytics.
